## ESC AM32 Configuration
1. Download configurator tool for linux (maybe older version)
1. Connect BEC to the tool, tool to the USB
1. Connect battery to the ESC
1. Run: `chmod +x SerialPortConnector.sh`
1. Run `./SerialPortConnector.sh`
1. Tick `direct connection`, choose port and click `Connect`
1. Click `M1` next to `Connect and Read Settings`
1. Wait a bit until list of settings will be loaded. 
1. Change what you want to change 
1. Click save
1. Reconnect to see if change is applied
